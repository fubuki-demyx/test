¿Que es la W3C?

El Consorcio de la World Wide Web es una comunidad internacional donde sus miembros se encargan de generar protocolos, estandares y modelos para asegurar el crecimiento a largo plazo del World Wide Web. Su finalidad es la creación de una Web universal, accesible y fácil de usar.

¿Que quiere decir HTML y cual es la version actual?

¿Cual es la función principal del portal http://www.w3schools.com?

La funcion pricipal es proveer tutoriales y ejemplos de código para elementos que se pueden crear con los lenguajes utilizados para la programación WEB.

¿Que formatos de imagen permite la Internet?

PNG, JPEG, GIF, SVG

¿Que formatos de video permite la Internet?

MPEG-4, AVI, DivX, MOV, FLV y WMV

¿Que formatos de audio permite la Internet?

WAV, MP3, OGG y MIDI
